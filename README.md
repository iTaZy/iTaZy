### 🏆 Projeto em evidência: HoK HuB

<p align="center">
  <img src="caminho/para/imagem-hokhub.png" alt="HoK HuB" width="80%" />
</p>

> Plataforma competitiva completa pra jogadores de Honor of Kings: matchmaking automático com MMR oculto, draft entre capitães, integração com Discord (canais de voz automáticos!), feed da comunidade e chat em tempo real.

**Stack usada:** React + Vite no front, Firebase (Firestore, Realtime Database, Cloud Functions, Auth) no back.

🔗 [Confira o repositório do HoK HuB](https://github.com/seu-usuario/hok-plataforma)
